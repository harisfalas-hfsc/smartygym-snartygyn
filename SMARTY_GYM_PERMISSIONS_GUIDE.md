# Smarty Gym - Comprehensive Permissions Guide

## Overview
This document outlines all user permissions across the Smarty Gym platform based on three user tiers:
1. **Visitors (Guests)** - Not logged in
2. **Logged-In Members (Subscribers)** - Authenticated users without premium subscription
3. **Premium Members** - Gold or Platinum subscription holders

---

## 1. VISITOR PERMISSIONS (Not Logged In)

### ✅ **Full Access**
- **Homepage** - Browse all features and subscription plans
- **Exercise Library** - Browse complete exercise database with videos and instructions (NO LOGIN REQUIRED)
- **Blog** - Read all fitness articles and educational content (NO LOGIN REQUIRED)
- **Coach Profile** - View Haris Falas's credentials and background
- **About Page** - Learn about Smarty Gym
- **Contact Page** - Submit contact form inquiries
- **Take a Tour** - Interactive platform walkthrough
- **Privacy Policy** - View privacy terms
- **Terms of Service** - View terms and conditions
- **Disclaimer** - View fitness disclaimer

### ⚠️ **Can Browse But Cannot Access**
- **Free Workouts** - Can see workout listings and details, but LOGIN REQUIRED to view full workout content
- **Free Training Programs** - Can see program listings and details, but LOGIN REQUIRED to view full program content

### ❌ **No Access**
- **Calculators** (BMR, 1RM, Macro Calculator) - Requires login
- **Community Page** - Requires login
- **Premium Workouts** - Requires premium subscription
- **Premium Training Programs** - Requires premium subscription
- **User Dashboard** - Requires login
- **Profile Settings** - Requires login
- **Workout Interactions** (favorites, ratings, completion tracking) - Requires premium
- **Diet Plan Generator** - Requires login
- **Personal Training Request** - Available but requires form submission

## Feature Interaction Details

### 1. Favorite System
**Who Can Use:** Premium Members Only

**How It Works:**
- Click heart icon on workout or program detail page
- Item is saved to "My Favorites" in dashboard
- Quick access from main navigation
- Can unfavorite at any time
- No limit on number of favorites

**Access Attempt by Non-Premium:**
- Shows upgrade prompt
- "Add this feature to your premium plan"
- Button redirects to subscription page

---

### 2. Rating System
**Who Can Use:** Premium Members Only

**How It Works:**
- 5-star rating system
- Available on each workout/program page
- Ratings are personal and contribute to overall average
- Can update rating at any time
- View your own rating history in dashboard

**Access Attempt by Non-Premium:**
- Shows upgrade prompt
- Stars are visible but not clickable
- Tooltip: "Upgrade to rate workouts"

---

### 3. Completion Tracking
**Who Can Use:** Premium Members Only

**How It Works:**
- "Mark as Complete" button on each workout/program
- Tracks completion date and time
- Shows completion percentage in dashboard
- Completion history available for review
- Contributes to progress analytics

**Access Attempt by Non-Premium:**
- Shows upgrade prompt
- Button disabled with lock icon
- Tooltip: "Premium feature - Track your progress"

---

### 4. View Tracking
**Who Can Use:** All Logged-In Users (Free + Premium)

**How It Works:**
- Automatically records when you view a workout/program
- No manual action required
- View history available in dashboard
- Helps recommend similar content
- Premium users get additional analytics

## Access Gate Behavior

### When Visitor Tries to Access Free Workout/Program Content:
1. Shows lock icon modal
2. Message: "Login Required - Please log in or create an account to access free workouts and training programs"
3. Two buttons:
   - "Log In / Sign Up" → Redirects to /auth
   - "Back to Home" → Redirects to homepage

### When Visitor Tries to Access Other Logged-In Content (Calculators, Community, Dashboard):
1. Shows lock icon modal
2. Message: "Login Required - Please log in or create an account to access this content"
3. Two buttons:
   - "Log In / Sign Up" → Redirects to /auth
   - "Back to Home" → Redirects to homepage

### When Logged-In Member Tries to Access Premium Content:
1. Shows crown icon modal
2. Message: "Premium Content - This [workout/program] is available exclusively to Gold and Platinum subscribers"
3. Two buttons:
   - "View Premium Plans" → Redirects to homepage subscription section
   - "Go Back" → Returns to previous page

### When Premium Member Accesses Content:
- No gates or modals
- Direct access to all content
- Full feature set available
- Progress tracked automatically

## Authentication Flow

### Sign Up Process (New Users):
1. Visit /auth page
2. Enter full name, email, password
3. Accept terms and conditions
4. Click "Sign Up"
5. Avatar setup dialog appears (optional)
6. Profile completion questionnaire
7. Redirected to dashboard
8. Email confirmation (auto-confirmed in dev)

### Login Process (Returning Users):
1. Visit /auth page
2. Enter email and password
3. Click "Login"
4. Redirected to dashboard
5. Session persists across page reloads

### Logout Process:
1. Click avatar in navigation
2. Select "Logout" from dropdown
3. Session cleared
4. Redirected to homepage
5. Toast confirmation: "Logged out successfully"
